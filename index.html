  <Div>
        O <Div> é um elemento HTML que serve como um contêiner genérico, podendo receber diferentes tipos de conteúdos (texto, imagens etc.).
         O <Div> é um elemento bastante utilizado para a construção de layouts de páginas web e pode facilmente ser estilizado com o uso do CSS.
